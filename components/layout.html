Layout (Grid System)

A responsive 12-column grid system based on Bootstrap for layout organization.

Import

import Container from "orion/lib/container";
import Row from "orion/lib/row";
import Col from "orion/lib/col";

Components

Container

The outermost container that provides a responsive or fluid wrapper for rows and columns.

Row

A horizontal group of columns.

Col

Individual columns for content placement.

Breakpoints

The grid system divides device screens into 5 categories:

Name Size Prop
Extra small (xs) Width < 576px xs
Small (sm) Width ≥ 576px sm
Medium (md) Width ≥ 768px md
Large (lg) Width ≥ 992px lg
Extra large (xl) Width ≥ 1200px xl

Container Props

Property Type Default Description
fluid Boolean false Makes the container full width instead of responsive
className String orion-container Custom CSS class

Row Props

Property Type Default Description
form Boolean false Reduces padding in columns for forms
className String - Custom CSS class
dataTestId String orion-row Custom data test ID for automation

Col Props

Property Type Default Description
xs Number | String | { size, order, offset } - Config for extra small devices (< 576px)
sm Number | String | { size, order, offset } - Config for small devices (≥ 576px)
md Number | String | { size, order, offset } - Config for medium devices (≥ 768px)
lg Number | String | { size, order, offset } - Config for large devices (≥ 992px)
xl Number | String | { size, order, offset } - Config for extra large devices (≥ 1200px)
className String - Custom CSS class
dataTestId String orion-col Custom data test ID for automation

Column Sizing, Ordering and Offsetting

<Container>
  <Row>
    <Col sm={{ size: 6, order: 2, offset: 1 }}>
      .col-sm-6 .order-sm-2 .offset-sm-1
    </Col>
  </Row>
  <Row>
    <Col sm="12" md={{ size: 6, offset: 3 }}>
      .col-sm-12 .col-md-6 .offset-md-3
    </Col>
  </Row>
  <Row>
    <Col sm={{ size: "auto", offset: 1 }}>.col-sm-auto .offset-sm-1</Col>

  </Row>
</Container>

Usage Notes

  • The grid system uses a 12-column layout, meaning columns can span from 1 to 12 units.
  • When no size is specified for a breakpoint, columns automatically distribute available space equally.
  • For responsive layouts, specify different column sizes for different breakpoints.
  • The size property in the object notation represents the column width (1-12 or "auto").
  • The order property changes the visual order of columns.
  • The offset property adds left margin to create space between columns.
  • Using xs="auto" makes the column width adjust to its content.
  • Columns automatically stack vertically on smaller viewports when larger breakpoints are specified.
