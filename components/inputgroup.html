InputGroup Component

A component that allows grouping of inputs with other controls or text elements.

Import

import InputGroup from "orion/lib/inputGroup";

Properties

Property Type Default Description
items Array - Array of component configurations to be rendered in the group. (Required)
disabled Boolean false Disables the entire input group.
className String - Provision to pass custom class.
dataTestId String "orion-input-group" Provision to pass a custom data test ID for automation purpose.

Items Array Structure

Each item in the items array should have the following structure: 

{
  component: String,  // Type of component: 'input', 'text', 'icon', 'button', or 'dropdown'
  data: Object        // Props specific to the component type
}

The data object varies based on the component type:

  • input: Accepts all Orion Input component props
  • text: { text: String, disabled: Boolean }
  • icon: React Element imported from orion-icons library
  • button: Accepts all Orion Button component props
  • dropdown: Accepts all Orion Dropdown component props

Note: At least one item with component: 'input' is required. Components placed before the input will be treated as prepend items, and those after will be append items.

Usage Examples

Text In Input Group

Text before input

@ 
<InputGroup
  items={[
    {
      component: "text",
      data: {
        text: "@"
      }
    },
    {
      component: "input",
      data: {
        placeholder: "Enter Email...",
        value: "orion@druva.org",
        onChange: () => {}
      }
    }
  ]}
/>

Text after input

$ 
<InputGroup
  items={[
    {
      component: "input",
      data: {
        type: "number",
        min: 0,
        placeholder: "Enter Amount..."
      }
    },
    {
      component: "text",
      data: {
        text: "$"
      }
    }
  ]}
/>

Text before and after input

http://orion.druva.org
@ 
<InputGroup
  items={[
    {
      component: "text",
      data: {
        text: "http://orion.druva.org"
      }
    },
    {
      component: "input",
      data: {
        type: "text",
        placeholder: "Enter Website..."
      }
    },
    {
      component: "text",
      data: {
        text: "@"
      }
    }
  ]}
/>
