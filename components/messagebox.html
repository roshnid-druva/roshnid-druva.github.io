MessageBox Component

A component for displaying messages with an icon, text, and optional actions.

Import

import MessageBox from "orion/lib/messageBox";

Properties

Property Type Default Description
size String "md" Size of message box: "lg", "md", or "sm".
primaryText String - Primary text of the message box.
secondaryText String - Secondary text of the message box (ignored in small message box).
icon React Element - Icon imported from orion-icons library. Required for large and medium box, optional for small. (Required)
action Object - Configuration for an action button or link.
className String - Custom CSS class.
dataTestId String "orion-message-box" Custom data test ID for automation.

Action Object Structure

When providing an action, use the following structure:

{
  type: String, // "button" or "link"
  data: {
    label: String,        // Text for the button or link
    onClick: Function,    // Click handler
    href: String,         // URL for link type
    disabled: Boolean     // Whether the action is disabled
  }
}

Size Variations

Large (lg)

  • Most spacious with larger icon
  • Good for empty states and important messages
  • Supports both primary and secondary text

Medium (md)

  • Default size
  • Balanced design for most use cases
  • Supports both primary and secondary text

Small (sm)

  • Compact design for limited space
  • Icon is optional
  • Secondary text is ignored

Usage Notes

  • Icons should be imported from the orion-icons library
  • For empty states in tables, use MessageBox as the noDataMessage prop
  • MessageBox is often used inside Card components for better visual organization
  • The component adapts its layout based on the size prop
