NumberText Component

A specialized component for displaying numeric text with consistent styling and various font weights and sizes.

Import

import NumberText from "orion/lib/numberText";

Properties

Property Type Default Description
as String "span" HTML element to render the component as
fontWeight Number 400 Font weight (300, 400, 500, 600, 700, 800)
fontSize String "md" Font size (xs, sm, md, lg, xl, 2xl, 3xl, 4xl, 5xl, 6xl)
className String - Additional CSS classes to apply
style Object - Inline styles to apply to the component
children ReactNode - Content to render inside the component

Font Size Options

The fontSize prop accepts the following values with corresponding sizes:

Value Size
xs 0.625rem
sm 0.75rem
md 0.875rem
lg 1rem
xl 1.125rem
2xl 1.25rem
3xl 1.375rem
4xl 1.5rem
5xl 1.75rem
6xl 2rem

Behavior Notes

  • NumberText applies a monospaced font style specifically designed for numeric content
  • It renders consistent number styles across the application
  • The component can be rendered as any HTML element using the as prop
  • Custom styles can be applied using the style prop or className
  • The component is useful for dashboards, financial data, statistics and anywhere consistent numeric presentation is needed
