ButtonGroup Component

A component to group multiple buttons together, supporting standard button groups, radio button groups, and checkbox button groups.

Import

import ButtonGroup from "orion/lib/buttonGroup";

Properties

Property Type Default Description
items Array - List of button objects to create the ButtonGroup. (Required)
selectedItems Array - An array of selected buttons name strings.
disabledItems Array - An array of disabled buttons name strings.
onItemClick Function - A callback function to handle the button click. Arguments event and item are passed to this function.
size String "sm" Specify the size of the button group. One of "sm", "md", "lg".
block Boolean false Forces the button group to occupy the whole parent width.
className String - Provision to pass custom class.
dataTestId String "orion-button-group" Provision to pass a custom data test ID for automation purpose.

Items Array Structure

Each item in the items array should have the following structure: 

{
  label: String,  // Label of the button (Required)
  name: String,   // Name of the button (Required)
  isSelected: Boolean, // Whether the button is selected
  disabled: Boolean,   // Whether the button is disabled
  onClick: Function    // Click handler for the button
}

Note: The ButtonGroup component does not handle the button selection to create Radio button group or checkbox button group. You have to manage the "isSelected" property of the button for such behavior.

Usage Examples

Standard Button Group

const items = [
  { label: "Left", name: "left" },
  { label: "Middle", name: "middle" },
  { label: "Right", name: "right" },
];

<ButtonGroup size="sm" items={items} />

Radio Button Group (Single Selection)

const radioItems = [
  { label: "Left", name: "left", onClick: this.onRadioBtnClick },
  {
    label: "Middle",
    name: "middle",
    onClick: this.onRadioBtnClick,
    isSelected: true,
  },
  { label: "Right", name: "right", onClick: this.onRadioBtnClick },
];

<ButtonGroup size="md" items={radioItems} />

Checkbox Button Group (Multiple Selection)

const checkboxItems = [
  { label: "Left", name: "left" },
  { label: "Middle", name: "middle" },
  { label: "Right", name: "right" },
];

const selectedCheckboxes = ["middle", "right"];

<ButtonGroup
  size="lg"
  items={checkboxItems}
  selectedItems={selectedCheckboxes}
  onItemClick={this.onChkboxClick}
/>

With Disabled Items

const calendarDays = [
  { label: "Su", name: "su" },
  { label: "Mo", name: "mo" },
  { label: "Tu", name: "tu" },
  { label: "We", name: "we" },
  { label: "Th", name: "th" },
  { label: "Fr", name: "fr" },
  { label: "Sa", name: "sa" },
];

<ButtonGroup
  items={calendarDays}
  disabledItems={["su", "fr", "we"]}
  selectedItems={["mo", "th", "fr"]}
/>

Block Button Group (Full Width)

const items = [
  { label: "Left", name: "left" },
  { label: "Middle", name: "middle" },
  { label: "Right", name: "right" },
];

<ButtonGroup block items={items} />
