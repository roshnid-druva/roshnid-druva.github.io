Label Component

A component to render form labels with consistent styling.

Import

import Label from "orion/lib/label";

Properties

Property Type Default Description
for String - ID of the form element that the label is bound to
text String - Label text content
optional Boolean false If true, displays "(Optional)" after the label text
sm Number - Grid column size for small screens (for responsive layouts)
md Number - Grid column size for medium screens
lg Number - Grid column size for large screens
xl Number - Grid column size for extra large screens
className String - Additional CSS classes to apply to the label
dataTestId String "orion-label" Data test ID for automation testing
children Node - Child elements to be rendered inside the label

Usage Examples

Basic Label

<Label for="username" text="Username" />

Optional Label

<Label for="bio" text="Biography" optional />

Label with Grid Column Size (Responsive)

<FormGroup row>
  <Label for="email" text="Email Address" sm={3} />
  <Col sm={9}>
    <Input type="email" id="email" name="email" />
  </Col>
</FormGroup>

Label with Custom Styling

<Label for="password" text="Password" className="oui-text-danger" />

Label with Children

<Label for="username">
  Username
  <span className="oui-text-muted oui-ml-1">(Required)</span>
</Label>

Label Used in a Form Group Row

<FormGroup row>
  <Label for="firstName" sm={2} text="First Name" />
  <Col sm={10}>
    <Input type="text" id="firstName" name="firstName" />
  </Col>
</FormGroup>
