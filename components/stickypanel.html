StickyPanel Component

A fixed position panel component that can be positioned at various corners of the screen, collapsed, dragged, and resized.

Import

import StickyPanel from "orion/lib/stickyPanel";

Components

StickyPanel

The root component that provides the container for the panel.

StickyPanel.Header

Used to render the header section of the sticky panel.

StickyPanel.Body

Used to render the main content section of the sticky panel.

StickyPanel.Footer

Used to render the footer section of the sticky panel.

StickyPanel Props

Property Type Default Description
children React Node Empty Content of the panel. (Required)
show Boolean false Controls the visibility of the sticky panel. (Required)
width Number 300 Width of the sticky panel in pixels
height Number 400 Height of the sticky panel in pixels
maxHeight Number 500 Maximum height of the sticky panel in pixels
close Function - Function to handle closing the panel
draggable Boolean false Enables dragging of the sticky panel
collapsible Boolean true Allows the sticky panel to be collapsed
defaultCollapsed Boolean false Whether the sticky panel should be collapsed by default
offsetX Number 10 Horizontal offset from the edge of the screen
offsetY Number 10 Vertical offset from the edge of the screen
position String bottom-right Position of the panel (top-left, top-right, bottom-left, bottom-right)
honourHeaderHeightOnResize Boolean true Whether to maintain the header height during resize
className String - Custom CSS class

StickyPanel.Header Props

Property Type Default Description
children React Node - Content of the header. (Required)

StickyPanel.Body Props

Property Type Default Description
children React Node - Content of the body. (Required)

StickyPanel.Footer Props

Property Type Default Description
children React Node - Content of the footer. (Required)

Usage Guidelines

  • The StickyPanel component is designed to display floating panels that can be positioned at different corners of the screen.
  • It's recommended to use the StickyPanel.Header, StickyPanel.Body, and StickyPanel.Footer components in sequence for a standard layout.
  • These subcomponents must be direct children of StickyPanel without any wrappers.
  • The first child is treated as the header, the second as the body, and the third (optional) as the footer.

Usage Examples

Basic Usage

Panel Header

✕

This is the panel content

import StickyPanel from "orion/lib/stickyPanel";
import Button from "orion/lib/button";

const StickyPanelExample = () => {
  const [show, setShow] = React.useState(true);
  const close = () => setShow(false);

  return (
    <>
      <Button
        label={show ? "Hide Panel" : "Show Panel"}
        onClick={() => setShow(!show)}
      />

      <StickyPanel
        width={400}
        height={300}
        show={show}
        close={close}
        position="bottom-right"
      >
        <StickyPanel.Header>
          <h4>Panel Header</h4>
        </StickyPanel.Header>
        <StickyPanel.Body>
          <p>This is the panel content</p>
        </StickyPanel.Body>
        <StickyPanel.Footer>
          <Button label="Action" />
        </StickyPanel.Footer>
      </StickyPanel>
    </>
  );
};

Behavior Notes

  • The StickyPanel can be positioned at any of the four corners of the screen.
  • When draggable is enabled, users can move the panel around the screen.
  • When collapsible is enabled, the panel can be collapsed to show only the header.
  • The close function controls the visibility of the panel and should update the show prop.
  • Panel dimensions can be controlled through width, height, and maxHeight props.
