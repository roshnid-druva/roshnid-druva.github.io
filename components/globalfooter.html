GlobalFooter Component

A non-sticky footer component that can be placed anywhere in the application with configurable links and copyright text.

Import

import GlobalFooter from "orion/lib/globalFooter";

Properties

Property Type Default Description
links Array [] Array of link objects to be rendered in the footer
copyrightText String - Copyright text to be displayed (© symbol added automatically)
className String - Custom CSS class for the footer
dataTestId String "orion-global-footer" Custom data test ID for automation

Link Object Properties

Each object in the links array can have the following properties:

Property Type Description
label String Text displayed for the link
href String URL for the link
target String Target attribute for the link (_blank, _self, _parent, _top)
renderComponent Function Function to render custom link component (e.g., NavLink)

Usage Examples

Basic Usage

Privacy Policy Support About Us
© 2023 Company Name
<GlobalFooter
  links={[
    { href: "/privacy", label: "Privacy Policy" },
    { href: "/support", label: "Support" },
    { href: "/about", label: "About Us" },
  ]}
  copyrightText="2023 Company Name"
/>

With Custom Link Rendering (e.g., for React Router)

Privacy Policy Support About Us
© 2023 Company Name
import { NavLink } from "react-router-dom";

<GlobalFooter
  links={[
    { href: "/privacy", label: "Privacy Policy" },
    {
      href: "/support",
      label: "Support",
      renderComponent: (link, key) => (
        <NavLink key={key} to={link.href}>
          {link.label}
        </NavLink>
      ),
    },
    { href: "/about", label: "About Us" },
  ]}
  copyrightText="2023 Company Name"
/>

With External Links

Privacy Policy Support
© 2023 Company Name
<GlobalFooter
  links={[
    {
      href: "https://example.com/privacy",
      label: "Privacy Policy",
      target: "_blank",
    },
    {
      href: "https://example.com/support",
      label: "Support",
      target: "_blank",
    },
  ]}
  copyrightText="2023 Company Name"
/>

Behavior Notes

  • The component displays links on the left side and copyright text on the right
  • The copyright symbol (©) is automatically prepended to the copyrightText
  • For single-page applications, use the renderComponent property to render router-specific links
  • When used with the Page component, it's recommended to use <Page.Footer> instead for proper sticky footer positioning
