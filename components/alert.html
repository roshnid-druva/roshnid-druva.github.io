Alert Component

A component for displaying important messages or notifications.

Import

import Alert from "orion/lib/alert";

Properties

Property Type Default Description
message String/Element - The message to be displayed inside the Alert component. (Required)
type String "danger" Type of alert message: "success", "warning", "danger", or "info".
closable Boolean false Makes the Alert closable with an (x) button.
onClose Function - Alert close handler function.
className String - Custom CSS class.
dataTestId String "orion-alert" Custom data test ID for automation.

Alert Types

Success Alert

User has been created successfully.
<Alert message="User has been created successfully." type="success" />

Warning Alert

You will not be able to recover the data once deleted.
<Alert
  message="You will not be able to recover the data once deleted."
  type="warning"
/>

Danger Alert

Error message
<Alert message="Error message" type="danger" />

Information Alert

The reset password link has been sent to your email.
<Alert
  message="The reset password link has been sent to your email."
  type="info"
/>

Alert Customization

Alert with a Link

An alert with an example link. Give it a click if you like.
<Alert
  message={
    <div>
      An alert with <a href="/somewhere">an example link</a>. Give it a click if
      you like.
    </div>
  }
/>

Alert with Custom HTML Structure

Oh no!

Something terribly went wrong. Contact admin to resolve the issue.

  • Error reason #1
  • Error reason #2
  • Error reason #3
<Alert
  message={
    <div>
      <h4 className="oui-alert-heading">Oh no!</h4>
      <p>Something terribly went wrong. Contact admin to resolve the issue.</p>
      <hr />
      <div className="oui-mb-0">
        <ul>
          <li>Error reason #1</li>
          <li>Error reason #2</li>
          <li>Error reason #3</li>
        </ul>
      </div>
    </div>
  }
/>

Events

Handling Alert Close Event

<Alert
  message="Error message"
  closable
  onClose={(e) => {
    // Handle alert close event
    console.log("Alert closed");
  }}
/>

Notes

  • The alert type changes the background color and border color of the alert
  • Set closable to true to show a close button that allows users to dismiss the alert
  • The message prop can accept plain text or complex JSX elements
  • When providing custom content, you can use classes like oui-alert-heading for styled headers
  • Always provide an onClose handler when setting closable to true
