Badge Component

A component to display informational labels with optional icons and interactive capabilities.

Import

import Badge from "orion/lib/badge";

Properties

Property Type Default Description
label String - Label to the badge. (Required)
style Object - A provision to customize the badge style. Currently only 'color' and 'backgroundColor' are customizable. Example: { color: "#935F5C", backgroundColor: "#F0DCDB" }
clickable Boolean true Flag to set the interactivity with the badge.
onClick Function - A function to handle badge click. Arguments badge props and event are passed to this function.
className String - Provision to pass custom class.
icon React Node - Icon imported from Orion Icons Library.
alignIcon String "left" Alignment of the icon with respect to the label.
dataTestId String "orion-badge" Provision to pass a custom data test ID for automation purpose.
active Boolean false Show the active state of the badge component.
variant "success" | "warning" | "danger" - Badge Variant. Specificity of variant will be higher than the style prop.

Usage Examples

Basic Badge

+ 4
<Badge label="+ 4" clickable={false} />

Clickable Badge with Icon

Show More ℹ️
<Badge
  icon={<InfoIcon inheritFontColor width={12} height={12} />}
  alignIcon="right"
  label="Show More"
  onClick={() => alert("Now I can show more details.")}
/>

Custom Styled Badge

Failure
<Badge
  label="Failure"
  clickable={false}
  style={{ color: "#935F5C", backgroundColor: "#F0DCDB" }}
/>
