Heading Component

A component for displaying headings with different levels of hierarchy.

Import

import Heading from "orion/lib/heading";

Properties

Property Type Default Description
type String | Number "6" Heading level from 1-6. By default it is 6.
text String - Text to be shown as the heading.
inline Boolean false When true, renders as a <span> to display adjacent to other elements.
className String - Provision to pass custom class.
dataTestId String "orion-heading" Provision to pass a custom data test ID for automation purpose.
