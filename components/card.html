Card Component

A component to create content containers with customizable titles, bodies, and footers.

Import

import Card from "orion/lib/card";

Properties

Property Type Default Description
title String|Function - Main title of the card. Can be a string or a function for custom rendering.
subtitle String - Secondary title of the card.
body Function - A callback function to render the card body. (Required)
footer Function - A callback function to render the card footer.
fullHeight Boolean false Makes cards in the same row have equal height regardless of content.
titleAction Function - Function to add action items like icons or buttons next to the card title.
collapsible Boolean false Allows user to collapse the card body.
collapsedByDefault Boolean false Card body will be collapsed on render.
onCollapse Function - Callback for card body collapsed.
onExpand Function - Callback for card body expanded.
className String - Provision to pass custom class.
dataTestId String orion-card Provision to pass a custom data test ID for automation purpose.
onClick Function - Click handler for the card (title, body, footer). Note: For collapsible cards, clicking on title or arrow icon will not trigger this handler.
disabled Boolean false Disables all card interactions with an overlay showing over it.
isLoading Boolean false Show loading bouncer icon when the component is in loading state.
isError Boolean false Show error message when the component is in error state.

Card.Title Props

Property Type Default Description
title String - Title string to be shown
className String - Provision to pass custom class.

Note: If the title is not provided, subtitle and titleAction will be ignored altogether.

Usage Examples

Basic Card

<Card
  title="Card title"
  body={() => {
    return <div>Card Body goes here.</div>;
  }}
/>

Card with Subtitle and Footer

<Card
  title="Card With Footer"
  subtitle="Card Subtitle"
  body={() => {
    return <div>Card Body goes here.</div>;
  }}
  footer={() => {
    return <a href="#">Click Here To View</a>;
  }}
/>

Collapsible Card

<Card
  title="Collapsible Card"
  collapsible
  body={() => {
    return <div>Card Body goes here.</div>;
  }}
/>

// Initially collapsed
<Card
  title="Collapsed by Default"
  collapsible
  collapsedByDefault
  body={() => {
    return <div>Card Body goes here.</div>;
  }}
/>

Cards with Equal Height

<Row>
  <Col>
    <Card
      title="Card 1"
      fullHeight
      body={() => <div>Content with varying length...</div>}
    />
  </Col>
  <Col>
    <Card title="Card 2" fullHeight body={() => <div>Short content</div>} />
  </Col>
  <Col>
    <Card
      title="Card 3"
      fullHeight
      body={() => <div>Medium length content...</div>}
    />
  </Col>
</Row>

Custom Card Title

        />

    </div>

  body={() => {
    return <div>Card Body goes here.</div>;
  }}
/>
