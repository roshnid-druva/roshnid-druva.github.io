Card Component

The Card component is a flexible container that can be used to display content in a clear and organized way. It supports various features like headers, footers, titles, subtitles, and collapsible content.

Properties

Property Type Default Description
title String|Function - Main title of the card. If passing as a function, use the Card.Title component and pass the title prop as a string to it. Note: If title is not provided, subtitle and titleAction will be ignored.
subtitle String - Secondary title of the card.
body Function - A callback function to render the card body. (Required)
footer Function - A callback function to render the card footer.
fullHeight Boolean false Set multiple cards to equal height in a row.
titleAction Function - Function to add action items like icons next to the card title.
collapsible Boolean false Allows user to collapse the card body.
collapsedByDefault Boolean false Card body will be collapsed on render.
onCollapse Function - Callback for card body collapsed.
onExpand Function - Callback for card body expanded.
className String - Custom class for styling.
dataTestId String orion-card Custom data test ID for automation.
onClick Function - Click handler for the entire card. Note: For collapsible cards, clicking title/arrow won't trigger this.
disabled Boolean false Disables all card interactions and shows an overlay.
isLoading Boolean false Shows loading bouncer icon when component is loading.
isError Boolean false Shows error message when component is in error state.

Usage Examples

Basic Card

import Card from "orion/lib/card";

<Card
  title="Card Title"
  body={() => (
    <div>Card content goes here</div>
  )}
/>

Card with Subtitle and Footer

<Card
  title="Card Title"
  subtitle="Card Subtitle"
  body={() => (
    <div>Card content goes here</div>
  )}
  footer={() => (
    <div>Footer content</div>
  )}
/>

Collapsible Card

<Card
  title="Collapsible Card"
  collapsible={true}
  collapsedByDefault={true}
  body={() => (
    <div>Collapsible content goes here</div>
  )}
  onCollapse={() => console.log('Card collapsed')}
  onExpand={() => console.log('Card expanded')}
/>

Card with Title Actions

<Card
  title="Card with Actions"
  titleAction={() => (
    <Button icon="edit" variant="link" />
  )}
  body={() => (
    <div>Card content with actions</div>
  )}
/>

Custom Card Title

<Card
  title={() => (
    <Card.Title title="Custom Title">
      {/* Additional title content */}
    </Card.Title>
  )}
  body={() => (
    <div>Card with custom title</div>
  )}
/>

Styling

The Card component uses the following CSS classes for styling:

  • oui-card-collapsible - Applied to collapsible cards
  • oui-card-expanded - Applied when card is expanded
  • oui-card-collapsed - Applied when card is collapsed
  • oui-card-title - Styles for the card title
  • oui-card-title-container - Container for card title and actions
  • oui-card-title-action - Styles for title actions
