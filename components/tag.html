Tag Component

A component for displaying tag/label elements, optionally with closable functionality.

Import

import Tag from "orion/lib/tag";

Properties

Property Type Default Description
value String - Tag value. (Required)
label String - Optional label for the tag.
closable Boolean true Whether the tag is closable or not.
onClose Function - Function called when tag is closed. Receives tag props as argument.
className String - Custom CSS class.
dataTestId String "orion-tag" Custom data test ID for automation.
icon JSX - React element from orion icons library to display as an icon.
variant String - Tag variant: "success", "warning", or "danger". Takes precedence over style prop.

Usage Examples

Basic Tag

document × 
<Tag 
  value="document" 
  onClose={(tag) => alert(`closing tag '${tag.value}'`)} 
/>

Non-closable Tag

document 
<Tag value="document" closable={false} />

Tag with Label

type: folder × 
<Tag
  value="folder"
  label="type"
  onClose={(tag) => alert(`closing tag '${tag.value}'`)}
/>

Tag with Icon

⚙️ document × 
import SettingsBlue from "@druva-apps/orion-icons/lib/SettingsMultitoneFilled";

<Tag
  icon={<SettingsBlue />}
  value="document"
  onClose={(tag) => alert(`closing tag '${tag.value}'`)}
/>

Behavior Notes

  • When closable is set to true, a close (x) button will appear
  • The onClose handler receives the tag's props as an argument
  • If both icon and label are provided, the icon will appear before the label
  • Use the variant prop to apply predefined color schemes
