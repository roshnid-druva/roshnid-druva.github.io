ToggleButton Component

A component for toggling between on and off states, optionally with a label.

Import

import ToggleButton from "orion/lib/toggleButton";

Properties

Property Type Default Description
name String - Name of the toggle button.
isActive Boolean false Toggle button state (on/off).
isInvalid Boolean false Show toggle label as invalid. Useful for required toggles in forms.
onClick Function - Callback when toggle is clicked, receives (name, isActive, event).
label String - Label text to display next to the toggle.
disabled Boolean false Disable the toggle button.
className String - Custom CSS class for the toggle button container.
dataTestId String "orion-toggle-button" Custom data test ID for automation.

Usage Examples

Simple Toggle Button

constructor(props) {
  super(props);
  this.state = {
    isActive: false
  };
}

onClick = () => {
  this.setState({ isActive: !this.state.isActive });
}

render() {
  return (
    <ToggleButton
      isActive={this.state.isActive}
      onClick={this.onClick}
    />
  );
}

Behavior Notes

  • The component maintains its own visual state but requires the parent component to manage the actual state
  • The onClick handler provides the new state value as its second parameter
  • Setting isInvalid to true will style the label in an error state
  • Use the name prop for form identification or when managing multiple toggles
  • The toggle transitions smoothly between states with a sliding animation
  • Clicking either the toggle switch or the label will trigger the onClick callback
  • The disabled state preserves the toggle's current position but prevents interaction
