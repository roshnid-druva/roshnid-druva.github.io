Tooltip Component

A component that displays additional information when hovering over an element.

Import

import Tooltip from "orion/lib/tooltip";

Properties

Property Type Default Description
text String/Function - Text to be shown in the tooltip. A callback function can also be provided to show a custom tooltip. (Required)
placement String "top" Tooltip placement position: "auto", "top", "right", "bottom", "left"
textAlign String "left" Text alignment: "left", "center", "right" (Not applicable to custom tooltips)
maxWidth String/Number 200 Maximum width of the tooltip (Not applicable to custom tooltips)
overflowOnly Boolean false Show tooltip only when the tooltip target has overflow
showToolTip Boolean true Conditionally control tooltip visibility
className String - Additional CSS classes for the tooltip (applied to the inner wrapper div)
dataTestId String "orion-tooltip" Data test ID for automation testing

Custom Tooltip Content

You can use a function to return custom JSX for the tooltip content.

Custom Tooltip
Hi  I am
custom tooltip. 
<Tooltip
  text={() => (
    <div className="oui-text-center">
      <strong>Hi</strong> 
      <i>I am</i>
      <br />
      <span className="oui-text-danger">custom tooltip.</span>
    </div>
  )}
  placement="top"
>
  <a href="/">tooltip</a>
</Tooltip>

Behavior Notes

  • For non-custom tooltips (when text is a string), you can control text alignment and max width using the textAlign and maxWidth props.
  • When using custom tooltips (when text is a function), you need to handle the styling within your custom component.
  • The overflowOnly prop is useful when you want to show tooltips only for truncated text.
  • You can conditionally control tooltip visibility using the showToolTip prop.
  • Tooltips are triggered on hover and will automatically position themselves based on the placement prop.
  • For accessibility, tooltips are also shown when the element receives keyboard focus.
  • The tooltip's position will adjust automatically if it would otherwise be rendered outside the viewport.
