ActionButtons Component

A component for displaying a group of action buttons that can be configured to show some buttons directly and others in a dropdown menu.

Import

import ActionButtons from "orion/lib/actionButtons";

Properties

Property Type Default Description
buttons Array - List of button objects to render into action buttons component. (Required)
allInMore Boolean false Show all buttons in a dropdown menu instead of showing the first one separately
rightAligned Boolean true Controls the opening position of the dropdown menu
moreBtnSize String "md" Size of the more button ("sm", "md", "lg")
disableMoreBtn Boolean false Disable the more button that shows the dropdown
className String - Custom CSS class
dataTestId String "orion-action-buttons" Custom data test ID for automation

Button Object Properties

Property Type Description
label String Button label or dropdown menu item label
type String Button type ("primary" or "secondary")
size String Button size ("sm", "md", "lg")
disabled Boolean Flag to disable the button
onClick Function Click handler for the button or dropdown menu item

Usage Examples

With Three Buttons

const buttons = [
  { label: "Create", type: "primary" },
  { label: "Edit" },
  { label: "Delete" },
];

<ActionButtons buttons={buttons} />;

By default, the first button will be displayed directly, and the rest will appear in a dropdown menu.

With Two Buttons

const buttons = [
  { label: "Create", type: "primary" },
  { label: "Edit", type: "secondary" },
];

<ActionButtons buttons={buttons} />;

With One Button

const buttons = [{ label: "Create", type: "primary" }];

<ActionButtons buttons={buttons} />;

All Buttons in Dropdown Menu

const buttons = [
  { label: "Create", type: "primary" },
  { label: "Edit" },
  { label: "Delete" },
];

<ActionButtons buttons={buttons} allInMore rightAligned={false} />;

Behavior Notes

  • By default, the first button is displayed directly, and any additional buttons appear in a dropdown menu
  • Setting allInMore to true places all buttons in the dropdown menu
  • The dropdown menu is right-aligned by default but can be set to left-aligned using rightAligned={false}
  • Each button can have its own type, size, disabled state, and click handler
  • You can disable individual buttons by setting the disabled property to true in the button object
