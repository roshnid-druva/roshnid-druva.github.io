Divider Component

A component to visually separate content in lists or layouts with a horizontal or vertical line.

Import

import Divider from "orion/lib/divider";

Properties

Property Type Default Description
label String - Label to be put in middle of the divider.
orientation "horizontal" | "vertical" "horizontal" The orientation of the divider
className String - Provision to pass custom class.
dataTestId String "orion-divider" Provision to pass a custom data test ID for automation purpose.
