TrialBadge Component

A component that displays trial status information, including progress tracking, steps to complete, and trial expiration details.

Import

import TrialBadge from "orion/lib/TrialBadge";

If you only want to show the trial groups without the trial status, you can use the TrialProgress component: 

import TrialProgress from "orion/lib/TrialProgress/TrialProgress";

Properties

Property Type Default Description
variant String "minimal" Display style: "full", "minimal", "product", or "badgeOnly"
placement String "bottom-end" Popover placement (same as Popover component)
helpText String "Your trial status will appear here." Help text shown in the badge (pass null to hide)
trialTotalDays Number - Total days of the trial
trialRemainingDays Number 0 Remaining days of the trial (Required)
trialExpiryDate String - Formatted expiry date of the trial (Required)
onUpgrade Function - Click handler for upgrade button (shown when trial expired)
onOpen Function - Callback called when tooltip mounts
onClose Function - Callback called when tooltip unmounts
isLoading Boolean false Shows a loader instead of groups
hideGroups Boolean false Hides groups and trial message, shows only progress bar
groups Array - Array of trial progress groups
products Array - Array of products for the product variant

Group Object Structure

Each item in the groups array can have the following properties: 

{
  icon: ReactElement,           // Icon to display next to group name
  name: String,                 // Group name/title
  tooltip: String,              // Tooltip text for the group
  isCompleted: Boolean,         // Whether this group is completed
  onTitleClick: Function,       // Click handler for group title
  defaultExpand: Boolean,       // Whether steps should be expanded by default
  supressNextLabel: Boolean,    // Hide the "Next" label
  steps: [                      // Array of steps within this group
    {
      name: String,             // Step name
      isCompleted: Boolean,     // Whether step is completed
      tooltip: String           // Tooltip for the step
    }
  ]
}

Behavior Notes

  • The TrialBadge component is designed to guide users through the trial experience by showing their progress and next steps.
  • The "full" variant shows both the trial progress bar and the groups/steps to complete.
  • The "minimal" variant shows only the trial progress bar.
  • The "product" variant is designed to show product icons rather than detailed steps.
  • The "badgeOnly" variant shows just the trial badge without any additional information.
  • When the trial expires (trialRemainingDays is 0), it shows an expired state and can display an upgrade button.
  • The component automatically highlights the next group/step to complete with a "Next" label (unless suppressed).
  • Group titles can be made clickable by providing an onTitleClick handler, useful for navigation to setup pages.
  • Steps within groups can be expanded/collapsed to save space.
