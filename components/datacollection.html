DataCollection Component

Overview

The DataCollection component is a composite Orion component that provides a complete solution for rendering tables with advanced features like search, filter, and action buttons. It's perfect for building listing pages with rich functionality.

Import

import DataCollection from "orion/lib/dataCollection";

Usage with DynamicChildHeight

The DataCollection component requires a height to render properly. Use it with the DynamicChildHeight component: 

import DynamicChildHeight from "../../../components/helperComponents";

const MyComponent = () => {
    return (
        <div className="table-container">
            <DynamicChildHeight childElmClassName="table-list">
                <div className="table-list">
                    <DataCollection {...props}/>
                </div>
            </DynamicChildHeight>
        </div>
    );
}

Props

Main Props

Property Data Type Default Value Description
dataGridConfig Object See description Accepts all props of DataGrid component except Children, defaultExpandedRows, expandedRows, and onRowExpand. Includes configuration for pagination, table ID, responsiveness, etc.
actionButtonConfig Object - Configuration for action buttons including primary, group, and dropdown actions.
autoSuggestConfig Object - Configuration for search functionality including search data, callbacks, and customization options.
filterButtonConfig Object - Configuration for filter functionality including filter title, filters array, and callbacks.
paginationConfig Object - Configuration for pagination including page change handling and records per page.
listCountWrapperConfig Object - Configuration for the list count wrapper including refresh functionality and count display options.

DataGrid Configuration

Property Data Type Default Value Description
tableId String data-grid Unique identifier for the table
columnsDef* Array - Array of column definitions
data* Array - Array of row data items
uniqueDataKey String id Property used to uniquely identify each row
responsive Boolean false Enables responsive table behavior

Column Definition Properties

Property Data Type Default Value Description
key* String - Unique identifier for the column
title* String - Column header title
dataKey String - Key to access column value from row data
cellRenderer Function - Custom renderer for cell content
