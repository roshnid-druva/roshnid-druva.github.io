DataCollection Component

DataCollection is a powerful compound component that combines several atomic components into a single unit. It provides a complete solution for displaying and managing tabular data with features like filtering, searching, sorting, pagination, and action buttons.

Installation and Imports

Basic Import

import DataCollection from "orion/lib/dataCollection";
Height Requirement: DataCollection component needs to be wrapped in a container with a defined height to render properly. This is typically achieved by wrapping it in a DynamicChildHeight component from the Orion library.

Configuration Properties

Property Type Description
dataGridConfig Object Configuration for the data grid component. Key properties include:
  • data: Array - Array of data objects to display
  • columnsDef: Array - Column definitions array with properties:
    • field: string - Data field to display
    • title: string - Column header text
    • width?: number - Optional column width
    • cellRenderer?: Function - Custom cell rendering function
    • rowSpan?: Function - Function to calculate row span
    • sortable?: boolean - Enable column sorting
    • headerRenderer?: Function - Custom header rendering
  • tableId: string - Unique identifier for the table
  • multiSelect: boolean - Enable row selection
  • isExpandable: boolean - Enable row expansion
  • isSelectable: boolean - Enable row selection
  • enableExportToCsv: boolean - Enable CSV export
  • csvFileName?: string - Name for exported CSV file
  • enableConfiguration: boolean - Enable table configuration
  • enableCSVToastMessage?: boolean - Show toast on CSV export
  • enableFullScreen: boolean - Enable full screen mode
  • paginationEnabled: boolean - Enable pagination
  • enableRowSpan: boolean - Enable row spanning
  • tableHeight: string - Table height (e.g., "400px")
  • footerHeight?: number - Height of table footer
  • onSort: Function - Sort handler function
  • onLoadMore: Function - Infinite scroll handler
  • onRowClick: Function - Row click handler
  • onRowSelectionChange: Function - Selection change handler
  • onSelectAll: Function - Select all rows handler
  • emptyRenderer: Function - Custom empty state renderer
  • loadedAll: boolean - Whether all data is loaded
  • isLoading: boolean - Show loading state
  • isLoadingMore: boolean - Show loading more state
  • defaultSelectedRows: Array - Pre-selected rows
  • selectedRowsCountText: string - Text for selected rows count
  • responsive: boolean - Enable responsive behavior
  • enablePersistConfig: boolean - Persist table configuration
  • sortBy: string - Default sort field
  • sortOrder: "asc" | "desc" - Default sort order
  • isLoadingTotalRowsCount: boolean - Loading state for total count
  • totalRowsCount: number - Total number of rows
  • tableRef: React.RefObject - Reference to table element
actionButtonConfig Object Configuration for action buttons. Properties include:
  • buttons: Array - Array of button configurations:
    • label: string - Button text
    • type: "primary" | "secondary" | "danger" | "link" - Button type
    • onClick: Function - Click handler
    • icon?: ReactElement - Optional icon
    • disabled?: boolean - Disable state
  • groupActionButtons?: Array - Buttons shown when rows are selected
  • dropdownActions?: Array - Actions shown in dropdown menu
filterButtonConfig Object Configuration for filter buttons. Properties include:
  • filters: Array - Array of filter configurations:
    • label: string - Filter label
    • options: Array - Filter options
    • type?: "single" | "multi" - Selection type
    • defaultValue?: any - Default selected value
  • onFilterChange: Function - Filter change handler
  • clearAllFilters: Function - Clear filters handler
autoSuggestConfig Object Configuration for search functionality. Properties include:
  • placeholder: string - Search input placeholder
  • onSearch: Function - Search handler function
  • suggestions: Array - Autocomplete suggestions
  • minChars?: number - Minimum characters to trigger search
  • debounceTime?: number - Debounce time in milliseconds
paginationConfig Object Configuration for pagination. Properties include:
  • onPageChange: (page: number) => void - Page change handler
  • recordsPerPage: number - Number of records per page
  • getInfoText: (from: number, to: number, total: number) => string - Custom text formatter
  • totalPages?: number - Total number of pages
  • currentPage?: number - Current active page
listCountWrapperConfig Object Configuration for record count display. Properties include:
  • totalRowsCount: number - Total number of records
  • selectedRowsCount?: number - Number of selected records
  • selectedRowsCountText?: string - Custom text for selection count
  • refreshData?: Function - Data refresh handler

Usage Examples

Basic DataCollection

import DataCollection from "orion/lib/dataCollection";

const dataGridConfig = {
  data: tableData,
  columnsDef: [
    { field: 'name', title: 'Name' },
    { field: 'email', title: 'Email' },
    { field: 'role', title: 'Role' }
  ],
  tableId: "users-table",
  multiSelect: true,
  isSelectable: true
};

<DataCollection
  dataGridConfig={dataGridConfig}
/>

Complete DataCollection Example

import DataCollection from "orion/lib/dataCollection";
import MessageBox from "orion/lib/messageBox";
import WarningIcon from "@druva-apps/orion-icons/lib/WarningYellowOutline";

const dataGridConfig = {
  data: tableData,
  columnsDef: [
    {
      field: 'name',
      title: 'Name',
      sortable: true,
      cellRenderer: (value) => <strong>{value}</strong>
    },
    {
      field: 'status',
      title: 'Status',
      width: 120,
      cellRenderer: (value) => (
        <span className={`status-${value.toLowerCase()}`}>{value}</span>
      )
    }
  ],
  tableId: "users-table",
  multiSelect: true,
  isSelectable: true,
  enableExportToCsv: true,
  csvFileName: "users-data",
  enableConfiguration: true,
  enableCSVToastMessage: true,
  responsive: true,
  tableHeight: "400px",
  emptyRenderer: () => (
    <MessageBox
      icon={<WarningIcon />}
      primaryText="No data available"
      secondaryText="Please add some data to view here"
    />
  ),
  onSort: (field, order) => handleSort(field, order),
  onLoadMore: () => loadMoreData(),
  onRowClick: (row) => handleRowClick(row),
  onSelectAll: (selected) => handleSelectAll(selected)
};

const actionButtonConfig = {
  buttons: [
    {
      label: "Add User",
      type: "primary",
      onClick: () => handleAddUser()
    }
  ],
  groupActionButtons: [
    {
      label: "Delete Selected",
      type: "danger",
      onClick: () => handleDeleteSelected()
    }
  ],
  dropdownActions: [
    {
      label: "Export All",
      onClick: () => handleExportAll()
    }
  ]
};

const filterButtonConfig = {
  filters: [
    {
      label: "Status",
      type: "multi",
      options: ["Active", "Inactive", "Pending"],
      defaultValue: ["Active"]
    },
    {
      label: "Role",
      type: "single",
      options: ["Admin", "User", "Guest"]
    }
  ],
  onFilterChange: (filters) => handleFilterChange(filters),
  clearAllFilters: () => handleClearFilters()
};

const autoSuggestConfig = {
  placeholder: "Search users...",
  minChars: 2,
  debounceTime: 300,
  onSearch: (query) => handleSearch(query),
  suggestions: searchSuggestions
};

const paginationConfig = {
  recordsPerPage: 20,
  currentPage: 1,
  totalPages: 10,
  onPageChange: (page) => handlePageChange(page),
  getInfoText: (from, to, total) => `Showing ${from} to ${to} of ${total} users`
};

const listCountWrapperConfig = {
  totalRowsCount: 200,
  selectedRowsCount: 5,
  selectedRowsCountText: "users selected",
  refreshData: () => handleRefreshData()
};

return (
  <DataCollection
    dataGridConfig={dataGridConfig}
    actionButtonConfig={actionButtonConfig}
    filterButtonConfig={filterButtonConfig}
    autoSuggestConfig={autoSuggestConfig}
    paginationConfig={paginationConfig}
    listCountWrapperConfig={listCountWrapperConfig}
  />
);

Notes and Best Practices

Pagination vs Infinite Loading: To use pagination instead of infinite loading, set paginationEnabled: true in the dataGridConfig.
Row Spanning: When using row spanning functionality:
  • Enable it with enableRowSpan: true
  • Provide a custom rowRenderer function
  • Note that row spanning won't work if estimatedRowHeight is set
Height Management:
  • Always wrap DataCollection with DynamicChildHeight for proper rendering
  • Ensure unique class names for childElmClassName to avoid conflicts
  • The container div should have a defined width (percentage or fixed)
Selection Management:
  • Use useSingleSelect for radio button selection behavior
  • Use useMultiselect for checkbox selection behavior
  • Both hooks provide clean APIs for selection state management
  • Selection hooks handle all edge cases and provide consistent behavior
  • Always provide a unique tableId for proper functionality
  • Use multiSelect with caution when row spanning is enabled
  • Implement proper error handling in data loading and action callbacks
  • Consider performance implications when dealing with large datasets
  • Use the listCountWrapperConfig to provide clear feedback about data status
