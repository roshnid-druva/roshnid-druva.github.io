Input Component

A versatile form control component that supports various input types such as text, password, checkbox, radio, number, and more.

Import

import Input from "orion/lib/input";

Properties

Property Type Default Description
type String "text" Type of input: "text", "password", "checkbox", "radio", "textarea", "number", "select", "submit", "button", "reset", "email"
id String - HTML ID attribute for the input element
name String - Name of the input field (required for forms)
value String/Number/Boolean - Value of the input field
placeholder String - Placeholder text for text-based inputs
label String - Text label for radio and checkbox inputs
disabled Boolean false Disables the input field
min Number - Minimum value for number inputs
max Number - Maximum value for number inputs
inline Boolean false Display checkbox or radio inputs inline
validationProps Object - Formik props for form validation
isInvalid Boolean false Whether the input field has validation errors
disableInputFeedback Boolean false Disable the display of validation feedback
className String - Additional CSS classes to apply to the input
dataTestId String "orion-input" Data test ID for automation testing
onChange Function - Function called when input value changes
options Array - Options for select inputs

Usage Examples

Text Input

<Input
  type="text"
  id="username"
  name="username"
  placeholder="Enter your username"
/>

Password Input

<Input
  type="password"
  id="password"
  name="password"
  placeholder="Enter your password"
/>

Textarea Input

<Input
  type="textarea"
  id="description"
  name="description"
  placeholder="Enter a description"
/>

Number Input

<Input
  type="number"
  id="quantity"
  name="quantity"
  min={0}
  max={100}
  placeholder="0"
/>

Checkbox Input

<Input
  type="checkbox"
  id="agreement"
  name="agreement"
  label="I agree to the terms and conditions"
/>

Radio Input

<Input
  type="radio"
  id="option1"
  name="options"
  value="option1"
  label="Option 1"
/>
<Input
  type="radio"
  id="option2"
  name="options"
  value="option2"
  label="Option 2"
/>

Button Inputs

// Submit button
<Input type="submit" label="Submit Form" />

// Reset button
<Input type="reset" label="Reset Form" />

// Regular button
<Input type="button" label="Click Me" />

Disabled Input

<Input
  type="text"
  id="readonly"
  name="readonly"
  value="This field is disabled"
  disabled
/>

Input with Validation

Username is required
<Input
  type="text"
  id="username"
  name="username"
  placeholder="Enter username"
  validationProps={formikProps}
  isInvalid={formikProps.touched.username && formikProps.errors.username !== undefined}
/>
<InputFeedback
  show={formikProps.touched.username && formikProps.errors.username !== undefined}
  message={formikProps.errors.username}
/>

Select Input

<Input
  type="select"
  id="country"
  name="country"
  options={[
    { label: "United States", value: "us" },
    { label: "Canada", value: "ca" },
    { label: "Mexico", value: "mx" },
  ]}
/>
