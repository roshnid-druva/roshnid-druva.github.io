Button Component

A versatile button component with multiple types, sizes, and states.

Import

import Button from "orion/lib/button";

Properties

Property Type Default Description
label String - The label of the button
type String secondary Type of the Button component. Can be either "primary", "secondary", "danger" or "link"
isSuccess Boolean false If passed as true, the button will be shown in success color. Using isSuccess with primary and secondary type will show filled and outlined buttons respectively.
size String md Size of the Button component. Can be either "sm", "md" or "lg". Font size, padding will be automatically adjusted.
href String - href is required for link button only. If href is provided then the button is created using <a> else using <button>
icon React Element - Icon React Element imported from orion-icons library. If you want the icon color changed on hover of the button, please pass inheritFontColor prop to the Icon component.
disabled Boolean false Handles the disability of the button.
active Boolean false Makes button style looks like active.
block Boolean false Full width button.
onClick Function - Click handler function for the button.
onMouseDown Function - Mouse down handler function for the button.
onMouseUp Function - Mouse up handler function for the button.
className String - Provision to pass custom class.
dataTestId String orion-button Provision to pass a custom data test ID for automation purpose.

Usage Examples

Button Types

Primary Button

<Button label="click me" type="primary" />

Secondary Button

<Button label="secondary" />

Danger Button

<Button label="danger" type="danger" />

Button with Icon and Label

<Button
  icon={<Alert inheritFontColor inheritFontSize />}
  label="Alarm"
  type="primary"
/>

Link Button

<Button label="link button" type="link" href="/dummylink" />

Button States

Disabled Primary Button

<Button label="click me" type="primary" disabled />

Disabled Secondary Button

<Button label="secondary" disabled />

Disabled Link Button

<Button label="link button" type="link" href="/dummylink" disabled />

<Button label="Large Button" type="primary" size="lg" />

Best Practices

  • Use primary buttons for main actions
  • Use secondary buttons for alternative actions
  • Use danger buttons for destructive actions
  • Use link buttons for navigation or less important actions
  • When using icons, ensure they are from the orion-icons library
  • Keep button labels clear and concise
  • Use appropriate button sizes based on their importance and context
