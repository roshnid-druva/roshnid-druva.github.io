LabelValue Component

A component to display data in a label-value pair format, commonly used in detail views and information panels.

Import

import LabelValue from "orion/lib/labelValue";

Properties

Property Type Default Description
label String - The label text to display (Required)
value String/Node - The value to display (can be text or a React element)
isLoading Boolean false Shows a loading indicator when true
isError Boolean false Shows error styling when true
fullWidth Boolean false Makes the component take up full width of its container
labelAlign String "left" Alignment of the label text ("left", "right", "center")
className String - Additional CSS classes to apply to the container
dataTestId String "orion-labelvalue" Data test ID for automation testing

Usage Examples

Basic Usage

Username
johndoe
<LabelValue label="Username" value="johndoe" />

With Custom Link as Value

Profile
View Profile 
<LabelValue
  label="Profile"
  value={
    <a href="#" className="oui-text-link">
      View Profile
    </a>
  }
/>
