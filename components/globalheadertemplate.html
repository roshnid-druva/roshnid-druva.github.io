GlobalHeaderTemplate Component

A component that provides a consistent global header interface across Druva products, including navigation, branding, and menu capabilities.

Import

import GlobalHeaderTemplate from "orion/lib/globalHeaderTemplate";

Properties

Property Type Default Description
appDataSet Object - Data containing product information, navigation structure, and URLs
brand Object - Configuration for brand logo and name
getAppNavMenu Function - Function that returns application-specific navigation menu items
getCommonNavMenu Function - Function that returns common navigation menu items
globalSidebarEnabled Boolean false Whether the global sidebar is enabled

Brand Object Properties

Property Type Description
name String Brand name
portalName String Name of the portal
href String URL to navigate to when clicking the logo
isActive Boolean Whether the brand is active
logoUrl String URL to the brand logo image

App Navigation Menu Item Types

The getAppNavMenu function should return an array of menu item objects, each with a component property that defines its type.

import GlobalHeaderTemplate from "orion/lib/globalHeaderTemplate";
import SettingsIcon from "orion-icons/lib/SettingsGreyOutline";
import DownloadIcon from "orion-icons/lib/DownloadGreyOutline";

// AppDataSet containing product information
const appDataSet = {
  productDisplayName: "Product",
  druvaoneurl: "https://console.example.org/dashboard",
  globalSidebarContent: {
    // Content for the global sidebar
    // (See documentation for structure)
  },
};

// Brand configuration
const brandConfig = {
  name: "Product",
  portalName: "Backup Service",
  href: "/",
  isActive: true,
  logoUrl: "https://example.org/logo.svg",
};

// Application navigation menu generator
const getAppNavMenu = (appDataSet) => {
  return [
    {
      component: "dropdown",
      data: {
        label: "Availability",
        items: [
          {
            label: "Backup",
            href: "#/backup-manage",
          },
          {
            label: "Restore",
            href: "#/restore-manage",
          },
        ],
      },
    },
    {
      component: "link",
      data: {
        href: "#/user",
        name: "Users",
      },
    },
    {
      component: "icon",
      data: {
        icon: <DownloadIcon />,
        title: "Downloads",
        tooltip: {
          text: "Downloads",
          placement: "bottom",
        },
        href: "#/download",
      },
    },
  ];
};

// Common navigation menu generator
const getCommonNavMenu = (appDataSet) => {
  return [];
};

// Component usage
<GlobalHeaderTemplate
  appDataSet={appDataSet}
  brand={brandConfig}
  getAppNavMenu={getAppNavMenu}
  getCommonNavMenu={getCommonNavMenu}
  globalSidebarEnabled={true}
/>;

Behavior Notes

  • The GlobalHeaderTemplate provides consistent navigation across Druva products
  • The component displays brand information on the left and navigation on the right
  • Navigation can include dropdowns, links, and icon buttons with tooltips and badges
  • When globalSidebarEnabled is true, the component displays a global navigation sidebar
  • The appDataSet contains product information and is used by the navigation generators
  • Icon menu items can display badges for notifications or status indicators
