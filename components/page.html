Page Component

A layout component that enforces consistent page structure across products, handling headers, footers, navigation, and content areas.

Import

import Page from "orion/lib/page";

Components

Page is composed of several sub-components:

  • Page - Root container
  • Page.Main - Main content container
  • Page.Alert - Displays alert messages
  • Page.Header - Container for page header elements
  • Page.Title - Container for page title elements
  • Page.TitleText - Text component for consistent page titles
  • Page.Actions - Container for header action buttons or elements
  • Page.Body - Main content area
  • Page.Tabs - Tab navigation for page content
  • Page.Footer - Sticky footer component

Page.Body

Property Type Default Description
className String - Additional CSS class for the body container
dataTestId String "orion-page-body" Custom data test ID for automation
scrollable Boolean true Whether the body content is scrollable

Page.Tabs

Property Type Default Description
items Array - Array of tab items to render (Required)
children ReactNode - Content when using tabbar mode
activeTabId String - ID of the currently active tab
tabbar Boolean false Use tabs as a navigation bar instead of tab panels
onTabClick Function - Handler function when a tab is clicked
className String - Additional CSS class for the tabs container
dataTestId String "orion-page-tabs" Custom data test ID for automation
borderBottom Boolean false Whether to show a border at the bottom of tabbar
variant String - Tab style variant ("default", "classic" or "widget")

Page.Alert

Wrapper around the Alert component. Accepts all Alert component props.

Page.Footer

Wrapper around the GlobalFooter component. Accepts all GlobalFooter component props.

Behavior Notes

  • The Page component manages the layout and enforces consistent spacing
  • The Page.Body component is scrollable by default and sizes automatically
  • When using Page.Tabs, tab content is managed automatically unless using tabbar mode
  • The Page.Footer component renders a sticky footer at the bottom of the page
  • Alert messages can be displayed either within Page.Main or outside for full-width alerts
