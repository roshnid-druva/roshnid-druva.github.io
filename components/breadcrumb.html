Breadcrumb Component

A navigation component that helps users understand their current location within a website's hierarchy.

Import

import Breadcrumb from "orion/lib/breadcrumb";

Properties

Property Type Default Description
items Array - List of objects to build the breadcrumb component. (Required)
onItemClick Function - A callback function to handle the breadcrumb click. Arguments event and item are passed to this function.
className String - Provision to pass custom class.
dataTestId String "orion-breadcrumb" Provision to pass a custom data test ID for automation purpose.

Items Array Structure

Each item in the items array should have the following structure: 

{
  label: String,      // The text label to be shown as a link in the breadcrumb
  href: String,       // URL to bind to the link in the breadcrumb (optional for last item)
  renderComponent: Function // A function to accept the router link if regular link is not expected
}

Usage Examples

Basic Usage

Home / Components / Breadcrumb 
const items = [
  {
    label: "Home",
    href: "/home",
  },
  {
    label: "Components",
    href: "/components",
  },
  {
    label: "Breadcrumb",
  },
];

<Breadcrumb items={items} />

With Click Handler

Home / Components / Breadcrumb 
const items = [
  {
    label: "Home",
    href: "/home",
  },
  {
    label: "Components",
    href: "/components",
  },
  {
    label: "Breadcrumb",
  },
];

<Breadcrumb
  items={items}
  onItemClick={(e, item) => {
    e.preventDefault();
    console.log(item.label + " clicked");
  }}
/>

With Custom Rendering (for React Router)

Home / Components / Breadcrumb 
import { NavLink } from "react-router-dom";

const items = [
  {
    label: "Home",
    href: "/about",
    renderComponent: (item) => <NavLink to={item.href}>{item.label}</NavLink>,
  },
  {
    label: "Components",
    href: "/accordion",
    renderComponent: (item) => <NavLink to={item.href}>{item.label}</NavLink>,
  },
  {
    label: "Breadcrumb",
  },
];

<Breadcrumb items={items} />
