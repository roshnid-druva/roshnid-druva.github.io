MotionDiagram Component

A component for rendering Lottie animations from Bodymovin JSON data.

Import

import MotionDiagram from "orion/lib/motionDiagram";

Properties

Property Type Default Description
src Bodymovin JSON data or URL to JSON - Animation source data. (Required)
loop Boolean false Whether to loop the animation
autoplay Boolean true Autoplay animation on load
speed Number 1 Control the animation playback speed
className String - Custom CSS class for the Lottie container

Methods

When using a ref on the MotionDiagram component, you can access these methods:

setLoop(boolean) - Enable or disable animation looping
setPlayerSpeed(number) - Change the animation playback speed
play() - Play the animation

Events

The MotionDiagram component can emit events that you can handle with the onEvent prop:

  • "complete" - Fired when the animation completes (and will not loop again)

Usage Examples

Basic Usage

import MotionDiagram from "orion/lib/motionDiagram";
import animationData from "./animation.json";

const BasicExample = () => {
  return <MotionDiagram src={animationData} autoplay loop />;
};

With Controls

Animation Speed
1 2 3 4 5 
import React, { useRef, useState } from "react";
import MotionDiagram from "orion/lib/motionDiagram";
import Slider from "orion/lib/slider";
import Input from "orion/lib/input";
import animationData from "./animation.json";

const ControlledExample = () => {
  const [loop, setLoop] = useState(false);
  const [animStopped, setAnimStopped] = useState(false);
  const lottieRef = useRef(null);

  const handleEvent = (event) => {
    if (event === "complete") {
      setAnimStopped(true);
    }
  };

  const handleLoopChange = (e) => {
    setLoop(e.target.checked);
    if (lottieRef.current) {
      lottieRef.current.setLoop(e.target.checked);
      if (animStopped) {
        lottieRef.current.play();
      }
    }
  };

  const handleSpeedChange = (value) => {
    if (lottieRef.current) {
      lottieRef.current.setPlayerSpeed(value);
    }
  };

  return (
    <div>
      <div>
        <label>
          Loop Animation
          <Input type="checkbox" checked={loop} onChange={handleLoopChange} />
        </label>

        <div>
          <label>Animation Speed</label>
          <Slider
            min={1}
            max={5}
            step={1}
            labelStep={1}
            tooltip
            onChange={handleSpeedChange}
          />
        </div>
      </div>

      <MotionDiagram
        ref={lottieRef}
        src={animationData}
        autoplay
        loop={loop}
        speed={1}
        onEvent={handleEvent}
        style={{
          height: "400px",
          width: "100%",
        }}
      />
    </div>
  );
};

Usage Notes

  • The MotionDiagram component is designed to work with Lottie animations exported from After Effects using the Bodymovin plugin.
  • The src prop can accept either a JSON object containing the animation data or a URL to a JSON file.
  • For large animations, consider loading the animation data asynchronously to improve performance.
  • You can control the animation programmatically using a ref to access the component's methods.
  • Animations can be styled with custom dimensions using inline styles or CSS classes.
