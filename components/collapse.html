Collapse Component

A component to show/hide content with expandable and collapsible sections.

Import

import Collapse from "orion/lib/collapse";

Properties

Property Type Default Description
trigger Function - Trigger to expand or collapse the body. Arguments onClick and isOpen are passed to the function.
isOpen Boolean false Toggle collapsible content.
body Function - Body content to be rendered. Argument isOpen is passed to this function. (Required)
onOpened Function - Function to trigger on expand of the content.
onClosed Function - Function to trigger on collapse of the content.
className String - Provision to pass custom class.
dataTestId String "orion-collapse" Provision to pass a custom data test ID for automation purpose.

Controlled Collapse

In controlled mode, you manage the open/closed state externally.

Expand

// In your component's state
this.state = {
  isOpen: false
};

// Toggle method
toggleState = () => {
  this.setState(prevState => ({
    isOpen: !prevState.isOpen
  }));
};

// Render method
<Button
  label={this.state.isOpen ? "Collapse" : "Expand"}
  onClick={this.toggleState}
/>
<Collapse
  isOpen={this.state.isOpen}
  body={() => (
    <Card
      className="oui-mt-2"
      body={() => (
        <div>
          Lorem Ipsum is simply dummy text of the printing and
          typesetting industry. Lorem Ipsum has been the industry's
          standard dummy text ever since the 1500s.
        </div>
      )}
    />
  )}
/>
