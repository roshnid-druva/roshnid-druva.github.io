Form Component

A component to create forms with validation support.

Import

import Form from "orion/lib/form";
import FormGroup from "orion/lib/formGroup";
import Input from "orion/lib/input";
import InputFeedback from "orion/lib/inputFeedback";
import Label from "orion/lib/label";

Form Props

Property Type Default Description
withValidation Boolean false Enable validation on the form, requires initialValues, validationSchema, and onSubmit
initialValues Object - Initial values for the form fields when using validation
validationSchema Function - Function that returns a Yup validation schema for form validation
onSubmit Function - Function called when form is submitted
renderChildren Function - Render function for form elements when using validation. Gets FormikProps as argument
className String - Provision to pass custom class
dataTestId String "orion-form" Provision to pass a custom data test ID for automation purpose

Form Components

FormGroup

A container for form controls with optional layout features.

Property Type Default Description
row Boolean false Display form elements in a horizontal row
footer Boolean false Style as a form footer with right-aligned content
className String - Provision to pass custom class
dataTestId String "orion-formgroup" Provision to pass a custom data test ID

InputFeedback

A component for displaying validation feedback for form controls.

Property Type Default Description
show Boolean - Whether to show or hide the feedback message
message String - The message to display
className String - Provision to pass custom class
dataTestId String "orion-input-feedback" Provision to pass a custom data test ID

Form with Validation

Required
Password must be at least 8 characters
Select at least one option
Must accept the terms and conditions 
<Form
  withValidation
  initialValues={{
    username: "",
    password: "",
    select: [],
    agreement: false,
  }}
  validationSchema={(Yup) =>
    Yup.object().shape({
      username: Yup.string()
        .min(2, "Too Short!")
        .max(50, "Too Long!")
        .required("Required"),
      password: Yup.string()
        .min(8, "Password must be at least 8 characters")
        .required("Required"),
      select: Yup.array()
        .min(1, "Select at least one option")
        .required("Required"),
      agreement: Yup.bool().oneOf(
        [true],
        "Must accept the terms and conditions"
      ),
    })
  }
  onSubmit={(values) => {
    console.log(values);
  }}
  renderChildren={(props) => (
    <React.Fragment>
      <FormGroup>
        <Label for="username" text="User Name" />
        <Input
          type="text"
          id="username"
          name="username"
          placeholder="Enter username here"
          validationProps={props}
          isInvalid={
            props.touched.username && props.errors.username !== undefined
          }
        />
        <InputFeedback
          show={props.touched.username && props.errors.username !== undefined}
          message={props.errors.username}
        />
      </FormGroup>

      {/* More form fields... */}
    </React.Fragment>
  )}
/>
