TimePicker Component

A component for selecting time from a dropdown list with customizable intervals.

Import

import TimePicker from "orion/lib/timePicker";

Properties

Property Type Default Description
value String - Time picker value. For 12-hour format: "12:00 AM" to "11:59 PM". For 24-hour format: "00:00" to "23:59".
interval Number 15 Time interval in minutes.
use24Hours Boolean false Show time picker in 24-hour format.
onTimeSelect Function - Callback when time is changed. Called only with valid time values.
isInvalid Boolean false Adds an error class to the time picker.
disabled Boolean false Disables the time picker.
allowCustomTime Boolean true Allows entering custom time. If false, invalid entries revert to previous value.
placeholder String - Custom placeholder text.
className String - Custom CSS class.
dataTestId String "orion-timepicker" Custom data test ID for automation.

Behavior Notes

  • The component supports both 12-hour and 24-hour time formats
  • When allowCustomTime is false, users can only select from predefined time options
  • The dropdown shows times at regular intervals defined by the interval prop
  • The component will validate time entries and revert to previous valid values if needed
  • The onTimeSelect callback is only called when a valid time is selected
  • When using a 12-hour format, the time includes AM/PM indicator (e.g., "09:15 AM")
  • When using a 24-hour format, the time is displayed without AM/PM (e.g., "09:15")
