PhoneInput Component

A specialized input component for international phone numbers with country code selection.

Import

import PhoneInput from "orion/lib/phoneInput";
Note: Please use the lib import shown above instead of named import because the underlying library has some issues with Orion build tooling. This issue will be resolved in later releases.

Properties

Property Type Default Description
name String - Input field name, useful for Formik forms
country String "us" Initial selected country
value String - Controlled input value
placeholder String - Custom placeholder text
disabled Boolean false Disables the input and dropdown
excludeCountries Array - Array of country codes to exclude from the dropdown
inputProps Object - Props to pass to the underlying input element
enableSearch Boolean false Enables search functionality in the country dropdown
onChange Function - Callback when value changes (value, country, event, formattedValue)
onBlur Function - Callback when input loses focus (event, country)
isInvalid Boolean false Marks the input as invalid (adds red border)
disableInputFeedback Boolean false Hides error messages during validation
validationProps Object - Formik validation props for form integration
dataTestId String "orion-phone-input" Custom data test ID for automation
direction String "down" Dropdown direction ("down" or "up")
className String - Custom CSS class

Behavior Notes

  • The component displays a dropdown menu for selecting country codes
  • When a country is selected, the appropriate country code is prefixed to the input
  • The dropdown can be configured to appear above or below the input using the direction prop
  • Search functionality can be enabled for finding countries more easily in the dropdown
  • The component integrates with Formik forms through the validationProps prop
  • Validation feedback can be customized or disabled as needed
  • The component supports both controlled and uncontrolled usage patterns
