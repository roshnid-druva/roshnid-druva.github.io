DatePicker Component

A simple calendar component for selecting dates without time or date range support.

Import

import DatePicker from "orion/lib/datePicker";

Properties

Property Type Default Description
name String - Name of the datepicker field. Useful for formik forms.
value Date/String/Number/null - Selected date value (native Date object, formatted string, or epoch time).
validationProps Object - Formik validation props for form integration.
disableInputFeedback Boolean false Prevents input error from showing when field is invalid.
onChange Function - Callback when date changes. Receives (nativeDate, formattedDate).
onFocus Function - Callback for input focus event.
onMonthChange Function - Callback when month changes. Receives (nativeDate, formattedDate).
onYearChange Function - Callback when year changes. Receives (nativeDate, formattedDate).
placement String "bottom-start" Placement of datepicker popper (auto-start, top, bottom, etc.).
isInvalid Boolean false Applies error styling to the input.
startYear Number 1990 First year shown in the year dropdown.
endYear Number Current Year Last year shown in the year dropdown.
dateFormat String "yyyy/MM/dd" Custom date format using date-fns compatible format.
openToDate Date Object - Force the datepicker to open at a particular date.
minDate Date Object - Minimum selectable date.
maxDate Date Object - Maximum selectable date.
highlightDates Array - Array of dates to highlight with a green bullet.
excludeDates Array - Array of dates to disable selection for.
customInput Function/Class - React component to render as custom input.
getInputRef Function - Function to get reference to the input element.
placeholder String - Placeholder text for the input field.
disabled Boolean false Disables the datepicker.
isClearable Boolean true Allows clearing the selected date.
className String - Custom CSS class for the calendar container.
dataTestId String "orion-datepicker" Custom data test ID for automation.

constructor(props) {
  super(props);
  this.state = {
    selectedDate: new Date()
  };
}

handleDateChange = (nativeDate, formattedDate) => {
  this.setState({
    selectedDate: nativeDate
  });
};

render() {
  return (
    <DatePicker
      value={this.state.selectedDate}
      onChange={this.handleDateChange}
      dateFormat="yyyy/MM/dd"
    />
  );
}

DatePicker with Custom Input

const CustomInput = ({ value, onClick, onChange, disabled }) => (
  <div className="custom-input-wrapper">
    <Input
      value={value}
      onClick={onClick}
      onChange={onChange}
      disabled={disabled}
      placeholder="Select a date"
      addon={<i className="di di-calendar" />}
    />
  </div>
);

<DatePicker
  value={this.state.selectedDate}
  onChange={this.handleDateChange}
  customInput={CustomInput}
  dateFormat="yyyy/MM/dd"
/>;

Behavior Notes

  • The DatePicker component works with native JavaScript Date objects internally
  • Input values can be provided as native Date objects, EPOCH timestamps, or correctly formatted date strings
  • When using custom input components, make sure to spread all provided props to the input element
  • The placement property will adjust automatically if there's not enough space in the specified direction
  • Highlighted dates display a green bullet beneath the date
  • Excluded dates are disabled and cannot be selected, even if they are also highlighted
  • For form integration, use the name and validationProps props with Formik
