Icon Component

A component for displaying icons, supporting both SVG icons and font-based icons.

Import

import Icon from "orion/lib/icon";

Properties

Property Type Default Description
src SVG Icon - Imported .svg file to be rendered as an image.
svg Inline SVG - Inline SVG to be rendered directly. Component renders the provided SVG as is.
classes String - Space-separated class names for font-based icons.
size String "sm" SVG icon size: "xs" (8x8), "sm" (16x16), "md" (24x24), "lg" (32x32).
width String - Custom width with unit (e.g., "24px"). Overrides the size property.
height String - Custom height with unit (e.g., "24px"). Overrides the size property.
actionable Boolean false Gives hover style to independent icons denoting actions (like reload). Best for font icons.
clickable Boolean false Adds hover effect to make icons appear interactive (like close icons).
muted Boolean true Default icon state. Gives opacity 0.55 to SVG and #737373 to font-based icons.
inactive Boolean false Applies styling to make the icon appear inactive.
onClick Function - Callback handler function for icon clicks.
className String - Provision to pass custom class.
dataTestId String "orion-icon" Provision to pass a custom data test ID for automation purpose.

Usage Examples

SVG Icons

Basic
Medium size
Custom size
Clickable

// Basic SVG icon
import HomeIcon from '@path/to/icons/home.svg';
<Icon src={HomeIcon} />

// SVG icon with size
<Icon src={HomeIcon} size="md" />

// SVG icon with custom dimensions
<Icon src={HomeIcon} width="24px" height="24px" />

// SVG icon with clickable state
<Icon src={HomeIcon} clickable onClick={() => console.log('Icon clicked')} />

// SVG icon with inactive state
<Icon src={HomeIcon} inactive />

Icon States

Icons can be displayed in different states:

1. Default (Muted)

This is the default state with reduced opacity

SVG icon
Font icon
<Icon src={HomeIcon} muted />
<Icon classes="di di-home" muted />

2. Clickable

Shows hover effects to indicate the icon is clickable

SVG icon
Font icon
<Icon src={HomeIcon} clickable />
<Icon classes="di di-home" clickable />

3. Actionable

Useful for action icons that stand independently

Refresh icon
Search icon
<Icon classes="di di-refresh" actionable />
<Icon classes="di di-search" actionable />

4. Inactive

Shows the icon in a disabled or inactive state

SVG icon
Font icon
<Icon src={HomeIcon} inactive />
<Icon classes="di di-home" inactive />

Notes

  • One Icon component can either serve an SVG icon (using src or svg) or a font icon (using classes), but not both simultaneously.
  • Set muted={false} when using colored SVG icons to prevent the default opacity from affecting the colors.
  • For font icons, consider using the actionable prop for icons that represent actions.
