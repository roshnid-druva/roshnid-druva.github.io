SideNav Component

A component for displaying hierarchical navigation menu in a sidebar, supporting collapsible groups, custom items, and icons.

Import

import SideNav from "orion/lib/sideNav";

Properties

Property Type Default Description
items Array - Array of navigation items to render. (Required)
onItemClick Function - Callback when a navigation item is clicked
className String - Custom CSS class for the SideNav container
dataTestId String "orion-sidenav" Custom data test ID for automation

Item Object Structure

The items prop accepts an array of objects with different possible structures:

Regular Link Item

{
  name: "Phoenix AWS Proxies",    // Text displayed for the item
  subtitle: "AWS Proxies",        // Optional subtitle
  href: "/aws_proxies",           // URL to navigate to
  target: "_blank",               // Optional target attribute for links
  extras: () => <NewTabIcon />,   // Optional function to render extra content
  icon: <PhoenixIcon />,          // Optional icon to display
  isActive: false,                // Whether the item is currently active
  disabled: false                 // Whether the item is disabled
}

Collapsible Group

{
  name: "DR Plans",               // Group name
  icon: <DisasterRecoveryIcon />, // Optional icon
  collapsible: true,              // Makes the group expandable/collapsible
  items: [                        // Child items
    { name: "Plan 1" },
    { name: "Plan 2" },
    { name: "Plan 3" }
  ]
}

Heading Group

{
  heading: "Resources",           // Heading text
  items: [                        // Items under this heading
    {
      name: "Documentation",
      href: "/docs",
      icon: <DocsIcon />
    },
    {
      name: "API Reference",
      href: "/api",
      icon: <ApiIcon />
    }
  ]
}

Custom Component Item

{
  renderComponent: (item, key, activeClass, otherClasses) => (
    <CustomComponent key={key} className={otherClasses} />
  ),
  collapsible: false,             // Optional, whether children should be collapsible
  items: [                        // Optional child items
    { name: "Child Item 1" },
    { name: "Child Item 2" }
  ]
}

Behavior Notes

  • Items can be regular links, collapsible groups, or custom components
  • Collapsible groups can be expanded or collapsed by clicking on the group header
  • When onItemClick is provided, it will be called with the item object and event when a clickable item is clicked
  • Active state can be controlled via the isActive property on items
  • Links with external URLs can use the target property to open in a new tab
  • Custom components can be rendered using the renderComponent property, which overrides other properties
  • The component supports hierarchical navigation with nested items
  • Heading groups are useful for organizing navigation into categories
  • Subtitles can be used to provide additional context for navigation items
  • Disabled items cannot be clicked and appear grayed out
  • The extras property allows adding small indicators to items, like external link icons
