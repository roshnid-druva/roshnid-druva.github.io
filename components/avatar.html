Avatar Component

A component to display user avatars with customizable variants and sizes.

Import

import Avatar from "orion/lib/avatar";

Properties

Property Type Default Description
label String - Label to the Avatar. (Required)
variant "success" | "warning" | "danger" | "info" "info" Avatar variant
size "sm" | "md" | "lg" "md" Avatar size
className String - Provision to pass custom class.
dataTestId String "orion-avatar" Provision to pass a custom data test ID for automation purpose.

Behavior Notes

  • The Avatar component automatically extracts the first letters from each word of the label to create the initials displayed in the avatar.
  • For single-word labels, it uses the first letter only.
  • For multi-word labels, it uses the first letter of the first and last words.
  • The size property controls the dimensions of the avatar, with "sm" being the smallest and "lg" being the largest.
  • The variant property changes the background color of the avatar, with "info" (blue) being the default.
  • Avatars are displayed as circles with centered text.
