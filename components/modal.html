Modal Component

A component that displays content in a layer above the page, typically used for dialogs, lightboxes, and other interactive features.

Import

import Modal from "orion/lib/modal";
import ConfirmationModal from "orion/lib/modal/confirmationModal";
import InformationModal from "orion/lib/modal/informationModal";

Modal Props

Property Type Default Description
isOpen Boolean false Controls whether the modal is displayed.
toggle Function - Function to toggle the modal state (required for closing on backdrop/escape/close button).
size String "md" Size of the modal: "sm", "md", "lg", "xl", "xxl" (full page).
header String/Node - Content for the modal header.
body Function - Render function for the modal body.
footer Function - Render function for the modal footer.
stickToHeader Boolean false When true, the modal sticks to the top navbar instead of being centered.
backdrop Boolean true Controls whether clicking on backdrop closes the modal.
keyboard Boolean true Controls whether pressing escape closes the modal.
zIndex Number 1050 CSS z-index value for the modal.
className String - Additional CSS classes for the modal.
contentClassName String - Additional CSS classes for the modal content.
docUrl String - URL for documentation link in the header.
docUrlTooltipText String - Tooltip text for the documentation link.
onOpened Function - Callback function called when the modal is fully opened.
onClosed Function - Callback function called when the modal is fully closed.
dataTestId String "orion-modal" Data test ID for automation testing.

ConfirmationModal Props

Property Type Default Description
isOpen Boolean false Controls whether the modal is displayed.
title String - Title for the confirmation modal.
message String - Primary confirmation message.
secondaryMessage String - Additional confirmation message.
continueButtonText String "Continue" Label for the primary action button.
cancelButtonText String "Cancel" Label for the cancel action button.
onContinue Function - Callback function when confirmation is accepted (required).
continueButtonType String "primary" Type of continue button: "primary" or "danger".
confirmReasonRequired Boolean false Shows a mandatory textarea for entering a reason.
reasonLabel String "Reason for deletion" Label text for the reason textarea.
reasonPlaceholder String "Please enter your comments here." Placeholder text for the reason textarea.
reasonMinLength Number 10 Minimum character length for the reason.
reasonMaxLength Number 150 Maximum character length for the reason.
emptyReasonError String "Please enter a reason." Error message when reason is empty.
reasonRegex RegExp - Regular expression to validate the reason text.
reasonRegexError String - Error message when reason doesn't pass regex validation.
invalidReasonLengthError String "Reason must be 10 to 150 characters long." Error message for invalid reason length.
toggle Function - Function to toggle the modal state (required).
closeOnEscape Boolean false Closes the modal when escape key is hit.
onOpened Function - Callback function called when the modal is opened.
onClosed Function - Callback function called when the modal is closed.
className String - Additional CSS classes for the modal.
dataTestId String "orion-confirmation-modal" Data test ID for automation testing.

InformationModal Props

Property Type Default Description
isOpen Boolean false Controls whether the modal is displayed.
message String - Primary information message.
secondaryMessage String - Additional information message.
body Function - Render function for modal body (overrides message props).
okButtonText String "Ok" Label for the primary action button.
toggle Function - Function to toggle the modal state (required).
closeOnEscape Boolean false Closes the modal when escape key is hit.
onOpened Function - Callback function called when the modal is opened.
onClosed Function - Callback function called when the modal is closed.
className String - Additional CSS classes for the modal.
dataTestId String "orion-information-modal" Data test ID for automation testing.

Usage Examples

Basic Modal

Modal Title

This is the modal content.

const [isOpen, setIsOpen] = useState(false);
const toggle = () => setIsOpen(!isOpen);

<Button label="Open Modal" onClick={toggle} type="primary" />

<Modal
  isOpen={isOpen}
  toggle={toggle}
  size="md"
  header="Modal Title"
  body={() => (
    <div>
      <p>This is the modal content.</p>
    </div>
  )}
  footer={() => (
    <React.Fragment>
      <Button label="Cancel" onClick={toggle} />
      <Button label="Save" type="primary" />
    </React.Fragment>
  )}
/>

Confirmation Modal with Required Reason

Confirm Deletion

Are you sure you want to delete this item?

Reason must be 10 to 150 characters long. 
<ConfirmationModal
  isOpen={isConfirmOpen}
  toggle={toggleConfirmation}
  title="Confirm Deletion"
  message="Are you sure you want to delete this item?"
  confirmReasonRequired={true}
  reasonLabel="Reason for deletion"
  reasonPlaceholder="Please explain why you're deleting this item"
  onContinue={(event, reason) => {
    console.log("Deletion reason:", reason);
    handleDelete();
  }}
/>
